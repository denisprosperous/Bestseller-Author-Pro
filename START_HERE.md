## üéØ Quick Start (5 Minutes)

### Step 1: Open the App

### Step 2: Add API Keys
1. Click **Settings** in the navigation

1. Click **Brainstorm**

5. Click "Use This Outline"
6. Click "Generate Book"
7. Wait 2-3 minutes
8. Export your ebook!

**Option B: Children's Book**
1. Click **Children's Ebooks**
2. Enter title and theme
3. Select age group and page count
4. Click "Generate Story"
5. Click "Generate Illustrations"
6. Export your book!

**Option C: Audiobook**
1. Click **Audiobooks**
2. Select an existing ebook
3. Choose voice provider
4. Generate audio
5. Export MP3!

## üí° Pro Tips

### For Best Results

**Text Ebooks**:
1. Use specific, detailed book ideas in Brainstorm
2. Choose GPT-4 or Claude 4 for best quality
3. Set word count to 30,000-40,000 for standard ebooks
4. Use "Professional" or "Conversational" tone
5. Specify your target audience clearly

**Children's Books**:
1. Keep themes simple and age-appropriate
2. Use 12-16 pages for picture books
3. Choose "Cartoon" or "Watercolor" style for young children
4. Use OpenAI or Leonardo AI for best illustrations
5. Review story before generating illustrations

**Audiobooks**:
1. Use ElevenLabs for most natural voices
2. Preview audio before generating full book
3. Choose voice that matches your content tone
4. Generate chapter by chapter for better control

## üêõ Troubleshooting

### Common Issues

**"No API key found"**
- Go to Settings and add your API key
- Make sure you're using the correct provider
- Check that your key is valid and has quota

**"Generation failed"**
- Verify your API key has sufficient quota
- Try a different AI provider
- Check your internet connection
- Refresh the page and try again

**"Export not working"**
- Try a different export format
- Check browser console for errors
- Make sure popup blockers are disabled
- Try a different browser

**Page not loading**
- Clear browser cache
- Check that dev server is running
- Verify URL is correct
- Check console for errors
